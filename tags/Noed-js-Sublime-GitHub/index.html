话说，做个测试看看效果如何吧：

前言\parbox{2em}{不搭后语} %\parbox设置宽度后，内容将自动换行。
\parbox和minipage可以带三个参数：t顶端 b底部 c居中
内容可以带四个参数：t顶端 b底部 c居中 s散开(s只在弹性间距生效)

前言\parbox[t]{2em}不搭后语
后语\parbox[b]{2em}不搭前言

\parbox[盒子位置][盒子高度][内容位置]{内容宽度}{盒子内容}

\begin{minipage}[盒子位置][盒子高度][内容位置]{内容宽度}
{盒子内容}
\end{minipage}

\begin{minipage}[c][2.5cm][t]{2em}
两个
\end{minipage}
\quad
\begin{minipage}[c][2.5cm][c]{3em}
黄鹂鸣翠柳
\end{minipage}
\quad
\begin{minipage}[c][2.5cm][b]{3em}
一行白鹭上青天
\end{minipage}
\quad
\begin{minipage}[c][2.5cm][s]{4em}
\setlength\parskip{0pt plus 1pt} %恢复正文默认段间距
窗含西岭千秋雪，\par
门泊东吴万里船。
\end{minipage}
